# Simple C Compiler

# 한국어

C++로 작성된 간단한 C 컴파일러.

본 리포지토리는 블로그[https://deepfield.blog/](컴파일러 포스팅의) 최종 결과물입니다.

타겟 아키텍처는 MIPS이고, 생성된 코드의 테스트는 SPIM을 통해 진행합니다.

주의: 코드를 정리하지 않은 상태로 업로드하는 것이라 아직 수정해야할 부분이 많이있습니다;;;
